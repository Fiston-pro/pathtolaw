# How to Add Your Video Files

Place your law video files in the `public/videos/` directory with the following names:

- `law1.mp4` - Prawo do pracy zdalnej
- `law2.mp4` - 4-dniowy tydzień pracy
- `law3.mp4` - RODO 2.0
- `law4.mp4` - Ban na plastik jednorazowy
- `law5.mp4` - Zwroty online - 30 dni
- `law6.mp4` - e-ZLA - Zwolnienia online
- `law7.mp4` - Darmowe podręczniki
- `law8.mp4` - Dotacje do fotowoltaiki
- `law9.mp4` - Strefa czystego transportu
- `law10.mp4` - Bezpłatne konto bankowe
- `law11.mp4` - Kredyt 2% dla młodych
- `law12.mp4` - Rodzina 800+
- `law13.mp4` - Aktywizacja zawodowa
- `law14.mp4` - Darmowy internet w szkołach
- `law15.mp4` - Zakaz alkoholu publicznie

The videos will be automatically loaded from this directory.

**Recommended video specs:**
- Format: MP4 (H.264)
- Resolution: 720p or 1080p (vertical format preferred, like 9:16)
- Duration: 15-60 seconds
- File size: Under 20MB for better performance
